# Teste To do List

## instalação do Projeto
```
1° Abra o projeto no seu VSCODE.
2° Abra o terminal do seu VSCODE, digite  'cd app' e tecle 'ENTER' para entrar na página de arquivos.
3° Em seguida digite 'cd api' e tecle 'ENTER', logo após digite 'json-server database.json' e tecle 'ENTER'.
4° Copie o código 'http://localhost:3000/todos' e cole numa aba do seu navegador, esse será seu banco de dados.
5° No seu VSCODE abra uma aba nova do seu terminal e repita o 2° passo.
6° Feito isso, digite o comando 'yarn serve' e tecle 'ENTER'.
7° Copie o código 'http://localhost:8080/' e abra o mesmo numa aba nova so seu computador.
8° Você terá acesso ao To-do-Lsit, onde após digitar a tarefa, você poderá atualizar o banco de dados e ver que sua tarefa criada está lá.
9° Você poderá marca-la como concluída, poderá editar a tarefa e excluí-la também.


```
